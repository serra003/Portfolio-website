## Tech Stack

Frontend:
- React (Vite)
- Tailwind CSS

Backend:
- Python (FastAPI / Flask)

Tools:
- Git & GitHub


## Features

- Interactive cybersecurity portfolio interface
- Modal-based case detail system
- Clean, recruiter-focused UI design
- Responsive layout

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```
### Backend
```bash
cd backend
python main.py
```
